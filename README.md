mq-vim
======

use vim-nox package instead of vim-tiny/vim-basic. It contains python and ruby bindings

for Command-T
-------------
```bash
rvm install 1.9.3
rvm use 1.9.3
cd ~/.vim/bundle/command-t/ruby/command-t
ruby extconf.rb
make
```

Adding plugins
--------------
```bash
git submodule add REPO_URL .vim/bundle/PLUGIN_NAME
```
